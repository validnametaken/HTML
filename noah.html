Games

A collection of fun and educational games
Color Quiz Guess what color is missing from the pictures.
Simon Says A memory challenge! Can you remember the color patterns? This is a nice game to practice the short-term memory and have a lot of fun. Can you get 10?
Number Speed Quick thinking! Picking the numbers from the lowest to highest is simple, but now it's on a timer. Can you get 10?
Block Drop The blocks are dropping. Put them in the right place!
Push Challenge A maze puzzle! Push colors to their correct location to complete the map. Sounds easy? Why don't you try it then.
Hiragana Quiz A small educational game to practice hiragana with the help of audio.
How Many Animals A fun activity for smaller children to count animals running across the screen.
Animal Quiz A fun activity for smaller children to recognize the sounds of different animals.
